# 2026-07-14 工作日志

## 集成测试代码编写

为 TrueUsed 二手交易平台后端项目编写了集成测试代码，使用 Testcontainers + Spring Boot 集成测试方案。

### 新增文件
- `TrueUsed/pom.xml` — 添加 spring-boot-testcontainers、testcontainers junit-jupiter、testcontainers mysql 依赖
- `TrueUsed/src/test/resources/application-test.properties` — 测试环境配置（JWT密钥、Flyway、占位服务等）
- `TrueUsed/src/test/java/com/xsh/trueused/testsupport/IntegrationTestBase.java` — 集成测试基类，包含 MySQL 8 + Redis 7 容器定义、MockMvc/ObjectMapper 注入、注册登录辅助方法
- `TrueUsed/src/test/java/com/xsh/trueused/integration/AuthIntegrationTest.java` — 认证模块集成测试（14个测试方法）
- `TrueUsed/src/test/java/com/xsh/trueused/integration/ProductIntegrationTest.java` — 商品模块集成测试（16个测试方法）
- `TrueUsed/src/test/java/com/xsh/trueused/integration/OrderIntegrationTest.java` — 订单模块集成测试（11个测试方法）

### 技术方案
- **Testcontainers + @ServiceConnection**：自动管理 MySQL 和 Redis 容器，Flyway 迁移自动执行
- **@SpringBootTest + @AutoConfigureMockMvc**：加载完整应用上下文，通过 MockMvc 发送 HTTP 请求
- **@ActiveProfiles("test")**：使用测试专用配置
- 每个测试方法使用 UUID 生成唯一用户名，避免数据冲突

### 测试覆盖
- 认证：注册、登录、JWT鉴权、参数校验、重复用户名、错误密码、权限控制
- 商品：CRUD全流程、上架/下架/擦亮、公开查询、搜索、跨用户权限校验
- 订单：完整生命周期（下单→支付→发货→确认收货）、订单查询、取消订单、自购限制、非参与人访问限制、物流查询

## GitHub Actions CI 配置

为项目创建了 GitHub Actions CI 流水线配置文件。

### 新增文件
- `.github/workflows/ci.yml` — CI 主流水线：后端单元测试、集成测试、打包、前端构建、Docker 镜像构建验证
- `.github/workflows/docker-publish.yml` — Docker 镜像发布到 GHCR（push main 或 tag 时触发，多架构 amd64+arm64）

### 修改文件
- `TrueUsed/pom.xml` — surefire 插件添加 `<excludedGroups>integration</excludedGroups>`，默认排除集成测试
- 三个集成测试类添加 `@Tag("integration")` 注解，支持按 tag 分组运行

### CI 架构
- **并行 Job**：backend-test（单元测试）、backend-integration（集成测试，需 Docker）、frontend-build、docker-build
- **串行依赖**：backend-package 依赖 backend-test，docker-build 依赖 backend-test + frontend-build
- **汇总检查**：ci-pass job 汇总所有 job 结果，可设为分支保护必过检查
- **本地运行单元测试**：`mvn test`（默认排除集成测试）
- **本地运行集成测试**：`mvn test -DexcludedGroups= -Dgroups=integration`
- **Docker 发布**：使用 GHCR（内置 GITHUB_TOKEN，零配置），支持 latest + 语义化版本 + SHA 标签，带 GHA 缓存
